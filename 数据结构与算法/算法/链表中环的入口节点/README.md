# 链表中环的入口节点

[提交链接](https://www.nowcoder.com/practice/6e630519bf86480296d0f1c868d425ad)

## 思路

- 先使用快慢指针找到第一个相遇结点。
- 然后再来一个指向第一个结点的指针，和相遇结点一起往下走，这两个指针相遇的结点就是环的入口结点。

<center>

![](list_circle.png)

</center>

如上图所示，$X$, $Y$, $Z$ 分别为链表起始位置，环开始位置和两指针相遇位置，则根据快指针速度为慢指针速度的两倍，可以得出：

$$
2 \cdot (a + b) = a + b + n \cdot (b + c)
$$

即：

$$
a = (n - 1) \cdot b + n \cdot c = (n - 1)(b + c) + c
$$

注意到 $b + c$ 恰好为环的长度，故可以推出，如将此时两指针分别放在起始位置和相遇位置，并以相同速度前进，当一个指针走完距离 $a$ 时，另一个指针恰好走出 绕环 $n - 1$ 圈加上 $c$ 的距离。
故两指针会在环开始位置相遇。
